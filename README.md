# composer-student-logs-provider
Gère les logs étudiants via syslog

Installation
------------

Modifier `composer.json` :

```
{
    // ...
    "require": {
        "etna/student-logs-provider": "~0.1"
    },
    "repositories": [
       {
           "type": "composer",
           "url": "http://blu-composer.herokuapp.com"
       }
   ]
}
```
